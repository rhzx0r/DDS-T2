Pasos para ejecutar el programa:
1: Compilar la clase App.java utilizando Javac desde la terminal. Ejemplo: javac App.java
2: Ejecutar App.class con java desde la terminal. Ejemplo: java App
3: Seguir las instrucciones del programa. 

* Nota *
